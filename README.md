# Ride Booking Analysis (OLA Case Study)
📌 Project Overview

This project analyzes a large-scale ride booking dataset containing 100,000+ records to understand customer behavior, operational efficiency, cancellation trends, and booking performance in a ride-hailing platform similar to OLA/Uber.

The analysis was performed using SQL, Excel, and Power BI to extract meaningful business insights and build KPI-driven dashboards.

🎯 Objectives
Analyze booking success and cancellation trends
Identify operational inefficiencies
Study customer and driver cancellation behavior
Evaluate ride demand patterns across locations and time
Measure booking value distribution and ride performance
Build business KPIs for decision-making
🛠️ Tools Used
SQL
Excel
Power BI

🔍 Analysis Performed
1. Booking Status Analysis

Analyzed the proportion of:

Successful rides
Cancelled rides by customers
Cancelled rides by drivers
Incomplete rides
Key Insight:

Successful rides accounted for approximately 62% of total bookings, while driver-side cancellations were significantly higher than customer cancellations.

2. Cancellation Analysis

Studied the primary reasons behind:

Customer cancellations
Driver cancellations
Customer Cancellation Reasons:
Driver not moving toward pickup
Driver asked to cancel
Change of plans
Wrong address
Driver Cancellation Reasons:
Personal & vehicle issues
Customer-related issues
Excess passengers
Key Insight:

Driver-related operational issues contributed heavily to overall cancellation rates.

3. Demand Pattern Analysis

Analyzed ride demand across:

Weekdays vs weekends
Different time periods
High-demand booking windows
Key Insight:

Weekend bookings and high-value rides increased significantly compared to weekdays.

4. Vehicle-Type Performance

Compared vehicle categories based on:

Booking volume
Revenue generation
Ride distance
Ratings
Key Insight:

Mini and Auto rides dominated booking volume, while Prime categories contributed higher booking values.

5. Revenue Analysis

Studied booking value distribution:

Low-value rides
Medium-value rides
Premium rides
Key Insight:

Most rides were concentrated in the lower booking-value segment, while premium rides contributed a smaller but higher-value share.

6. Ratings Analysis

Evaluated:

Driver ratings
Customer ratings
Relationship between service quality and ride completion
Key Insight:

Successful rides consistently showed higher customer and driver ratings.
